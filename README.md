# QANotz

**QANotz** (pronounced 'can-notes') is a small but powerful personal knowledge app built with Python and tkinter. It allows you to ask questions and give structured answers over time.

Write a question, then add answers as you find them. You can sort your answers in multiple ways, provide context, and refine them as you go.

---

## Why QANotz?

- **Editor/Viewer Switching:** Seamlessly switch between the editor and previewer with a hotkey and app button.
- **Readable Format:** Uses an easy to learn creation format for questions and answers.
- **Light and Powerful:** Small application and storage footprint.
- **Storage:** Uses a custom storage system designed around the format of QANotz.
